# Bittensor ( Version simple ET uniquement CPU ) 

## Installation 

Testé sur ubuntu 20.04 

### Prérequis

Super utilisateur
```sh
Sudo su
```

Mise à jours Ubuntu 
```sh
apt update
apt upgrade
cd
apt install python3-testresources
apt install python3-pip
```

Installation de Bittensor
```sh
pip3 install bittensor
```
